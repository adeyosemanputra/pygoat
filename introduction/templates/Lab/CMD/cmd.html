What is Command Injection

Command injection is an attack where the goal is execution of arbitrary commands on the host operating system via a vulnerable application. Command injection attacks are possible when an application passes unsafe user supplied data to a system shell.

The user is given a feature to perform a name server lookup. The domain is directly passed to the OS command without validation.

Exploiting

  • google.com && dir
  • google.com ; dir

This lab uses Python eval() to calculate expressions. Instead of math, you can run OS commands.

Example:

  • 1+1 → 2
  • os.system("id")


Mitigation

  • Validate all user input
  • Use whitelists
  • Avoid shell execution
  • Use safe APIs
