PyGoat - XSS (Cross-Site Scripting) Lab

⚠️ Educational Purposes Only
This lab intentionally contains security vulnerabilities for learning purposes.

About This Lab

This lab demonstrates Cross-Site Scripting (XSS) vulnerabilities. XSS is a common web vulnerability where attackers inject malicious scripts into web pages.

Vulnerability Type: Stored XSS (A03:2021 - Injection)

The application accepts user comments and displays them without proper HTML escaping. This allows attackers to inject JavaScript code that executes in other users' browsers.

Try the Lab

→ Go to Comments Lab

How to Exploit:

  1. Navigate to the Comments section
  2. Enter this payload in the comment box:
  3. <script>alert('XSS Vulnerability!')</script>
  4. Click "Post Comment"
  5. If a JavaScript alert appears, the vulnerability is confirmed!

Other Payloads to Try:

  • <img src=x onerror="alert('XSS')">
  • <svg onload="alert('XSS')">

How to Fix:

Remove the |safe filter in templates to enable HTML escaping.
