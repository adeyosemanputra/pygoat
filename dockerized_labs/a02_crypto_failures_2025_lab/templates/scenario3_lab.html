Attack Strategy

Since the token is generated using time.time() as the seed, you can reproduce the exact same token by:

  2. Seeding Python's random with that timestamp
  3. Calling the same random function

Lesson Learned:

Using weak PRNGs with predictable seeds (like timestamps) makes security tokens completely insecure. Always use cryptographically secure random number generators like Python's secrets module.
